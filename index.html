My Projects

LapStop

LapStop is a security web application for remote workers that enjoy working outside of their home. LapStop operates on the application's visibility, alerting users by e-mail when the application goes into a hidden state. The application also provides users with location and timestamp to track usage. Tech Stack: HTML5, CSS3, JavaScript ES6, jQuery, Socket.io, Node.js, JWT Authentication, MLab, Heroku

Check it out Live | Repo

Reddit Word Cloud Generator

A brainstorming application utilizing the Reddit API. Users input a search topic and a word cloud from the Reddit Hot Topic section will be generated. The words within the cloud appear largest based on frequency allowing users to find the most common topics. Users are able to continue their brainstorming by clicking on words within the cloud to produce a new cloud. Tech Stack: HTML5, CSS3, JavaScript ES6, jQuery

Check it out Live | Repo

GifComic

GifComic is a fun way for users to create comic strips using gifs. Users are able to choose the comic panel size, comic caption, number of comic panels and gif to create their own comic strip. Tech Stack: HTML5, CSS3, JavaScript ES6, Node.js, React.js
